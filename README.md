# Interactive Web Maps

Two interactive web map applications showcasing different mapping capabilities using Leaflet.js and OpenStreetMap.

## üåç Features

This project includes **two interactive maps**:

### Map 1: World Cities Map
- Displays 10 major cities around the world
- Interactive markers with detailed information
- Zoom and pan functionality
- Beautiful gradient purple theme
- Pop-up information panels for each city

### Map 2: Regional Explorer Map
- Focused on European cities and capitals
- Custom icons for capitals vs. major cities
- Interactive control buttons:
  - Show all locations
  - Focus on capitals
  - Toggle radius circles
  - Reset view
- Visual legend
- Pink/purple gradient theme
- Radius circles showing areas of influence

## üöÄ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, or Edge)
- No installation required - uses CDN for Leaflet.js

### Usage
1. Open `index.html` in your web browser to see the landing page
2. Click on either map card to explore:
   - **World Cities Map** (`map1.html`) - Global view with major cities
   - **Regional Explorer Map** (`map2.html`) - European cities with advanced features

### Live Demo
Simply open any of the HTML files directly in your browser:
- `index.html` - Landing page with navigation to both maps
- `map1.html` - World cities interactive map
- `map2.html` - European regional interactive map

## üìñ How to Use the Maps

### World Cities Map (Map 1)
1. Click on any marker to see city information
2. Use mouse wheel or +/- buttons to zoom
3. Click and drag to pan around the map
4. Click the instructions panel in the top-right for tips

### Regional Explorer Map (Map 2)
1. Use the control buttons at the top:
   - **Show All Locations** - Briefly displays all pop-ups
   - **Focus on Capitals** - Zooms to show all capital cities
   - **Toggle Radius Circles** - Shows/hides influence areas
   - **Reset View** - Returns to default zoom and position
2. Click markers to see detailed city information
3. Check the legend in the bottom-right corner
